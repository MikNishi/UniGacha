ДЛЯ РАБОТЫ ИСПОЛЬЗУЕМ ВЕТКУ develop - все нужные ветки от нее/доп.ветки - мержим тоже в нее/доп. ветку - main НЕ трогаем до релиза (что вряд ли случится...)
Использование дополнительных веток - приветствуется!
см. https://bitworks.software/2019-03-12-gitflow-workflow.html

1. Описание папок и для чего они нужны. Пожалуйста, старайтесь сохранять структуру папок. 
Если нужны дополнительные папки (чтобы разделить оружки/персов/артефакты) - создавайте, но в нужной директории.
Если какие-то папки считаете излишними - не используйте (смотри п. 4)

├── Core/                   # Базовые вещи проекта (CORE_)
│   ├── Scenes/             # Главные сцены (MainMenu.unity)
│   ├── Scripts/            # Менеджеры (GameManager.cs, SaveManager.cs и др.)
│   └── Prefabs/            # Общие префабы (MainCamera.prefab, EventSystemprefab)

├── World/                  # Основной 2D-изометрический мир (W_)
│   ├── Sprites/            # Спрайты зданий, персонажей, предметов (Audience01.png)
│   ├── Animations/         # Анимации для спрайтов (StudentWalk.anim)
│   ├── Tilemaps/           # Клетки карты, тайлы (не уверена, что будем использовать, но пусть будет)
│   ├── Prefabs/            # Префабы объектов мира (NPC, здания, предметы)
│   └── Scripts/            # Скрипты для логики мира (перемещение, взаимодействие) (NPCInteraction.cs)

├── VisualNovel/            # Сюжетная часть (VN_)
│   ├── Characters/         # Арты/спрайты персонажей (MainChar_Smile.png)
│   ├── Backgrounds/        # Фоны для сцен (DestroyedUniversity.png)
│   ├── Dialogues/          # Диалоги (JSON и т.п.) (intro_story.jon)
│   └── Scripts/            # Логика диалогов и переходов (DialogueManager.cs)

├── Roguelike/              # Мини-игра-рогалик (RL_)
│   ├── Sprites/            # 2D-графика рогалика (EnemyGoblin.png Hero)  
│   ├── Animations/         # Анимации врагов/героев (CoblinAttack.cs)
│   ├── Prefabs/            # Префабы объектов в рогалике
│   └── Scripts/            # Логика рогалика (бой, карты)

├── UI/                     # Общий UI проекта (UI_)
│   ├── Elements/           # Кнопки, панели, иконки
│   └── Scripts/            # Скрипты для интерфейса

├── Audio/                  # Аудио файлы (AU_)
│   ├── SFX/                # Эффекты
│   └── Music/              # Фоновая музыка

├── Fonts/                  # Шрифты для текста
├── Resources/              # Динамически загружаемые ресурсы (если нужно)
└── Plugins/                # Сторонние библиотеки и плагины


2. При создании файлов старайтесь добавлять префикс в начале названия. 
Например, префаб персонажа в рогалике: RL_char1.prefab, Core_Menu_Settings.prefab - префаб меню, RL_Ability_DoubleJump.cs - скрипт способности.
Чтобы было проще что-то искать и соотносить.

3. Не забывайте .meta файлы
4. Старайтесь не дублировать ассеты - если что-то общее, лучше поместить в Core.
5. Перед коммитом, на всякий случай, проверяйте, не закоммитили ли Library/Logs и тд.
